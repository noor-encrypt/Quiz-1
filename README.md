# Airbnb Clone Dashboard

A React-based Airbnb clone dashboard with user authentication, bookings, services, and profile management.

---

## Features

- User registration and login with token-based authentication
- Dashboard showing Home, Services, Bookings, Profile, About, and Contact sections
- Responsive UI using Tailwind CSS
- View and manage bookings
- Logout functionality

---

## Technologies Used

- React.js
- Tailwind CSS
- Express.js (Backend)
- MongoDB (Database)
- JWT (Authentication)
